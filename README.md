## 🤝 Contribuer

Les contributions sont les bienvenues ! Voici comment participer :

1. **Fork** le projet
2. Crée ta branche → `git checkout -b feature/ma-feature`
3. **Commit** tes changements → `git commit -m "feat: ma super feature"`

5. Ouvre une **Pull Request** 🔥
