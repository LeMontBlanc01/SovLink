## 👋 Hey, Bienvenue dans **SovLink**

**SovLink** est notre projet de messagerie en ligne — né avec une ambition claire :
fournir un moyen **simple, sécurisé et anonyme** de communiquer, sans compromis.

SovLink est une **application de messagerie chiffrée** dont l'objectif est simple :
permettre à plusieurs utilisateurs d'échanger des messages en temps réel via un serveur TCP — **sans compte, sans historique, sans trace**.

---

## Objectifs du projet

| Objectif | Description |
|:---:|:---|
| 🔐 | Fournir un système de **communication chiffrée de bout en bout** |
| 👤 | Garantir l'**anonymat des utilisateurs** à chaque connexion |
| ⚡ | Offrir une **messagerie temps réel** fluide et multi-clients |
| 🗑️ | Assurer **zéro stockage** des messages après transmission |
| 🔑 | Fonctionner **sans création de compte** ni identifiant fixe |

## 🔐 Sécurité

**SovLink repose sur 3 piliers :**

- **Chiffrement asymétrique** — chaque message est chiffré avec la clé publique du destinataire via **Libsodium**, seul le destinataire peut le déchiffrer
- **Identité éphémère** — aucun compte, aucun mot de passe ; une nouvelle paire de clés est générée à chaque connexion
- **Zéro mémoire** — les messages ne sont jamais stockés sur le serveur, ils disparaissent après transmission
